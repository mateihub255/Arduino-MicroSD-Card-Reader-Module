# Arduino-MicroSD-Card-Reader-Module
Schematic and PCB design for a functional MicroSD Card Reader.

### Specs:
* **Operating Voltage:** 4.5V - 5.5V DC
* **Current Requirement:** 0.2 - 200 mA
* **Voltage Regulator:** 3.3V on-board
* **File System:** Supports FAT
* **Storage:** Micro SD (up to 2GB) / Micro SDHC (up to 32GB)
